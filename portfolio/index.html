Мои работы

CarFinder

github.com

Приложение, которое позволяет аггрегировать все актуальные объявления по продаже автомобилей с крупнейших белорусских платформ. Приложение предоставляет фильтры для поиска и показа актуальных объявлений на основе пожеланий пользователя. Также можно сохранять параметры фильтров, чтобы получать уведомления по электронной почте о новых предложениях. Приложение также предоставляет возможность проверить ликвидность автомобиля по модели и типу кузова.

  • React
  • Redux
  • Bulma
  • KoaJS
  • MongoDB/Mongoose
